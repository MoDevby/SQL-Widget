# Shortcuts:
## General:
	* F1: Help.
	* F5: Execute.
	* Ctrl+Tab: Change the Tab.

## Select Tab:
	* Ctrl+Right: Add selected fields to the select clause.
	* Ctrl+Left: Remove selected fields from the select clause.

## Result Page:
	* F5: Re-Execute.
